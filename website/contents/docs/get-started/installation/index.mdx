Setting up AriaDocs is straightforward, with options to clone the repository or use the convenient `npx` command. AriaDocs requires [Node.js](https://nodejs.org/) version 18 or higher for optimal performance.

## Setup Environment
Follow the steps below to prepare your environment for using dose3.
<Stepper>
  <StepperItem title="NodeJS">
     This CLI is based on the Node.js npm package and written in JavaScript. You need to install Node.js first. <a href="https://nodejs.org/en/" target="blank">Click to download</a>
  </StepperItem>

  <StepperItem title="Git">
    This process uses a Git repository to clone the project. If you want to use it, you should install Git. <a href="https://git-scm.com/" target="blank">Click to download</a>
  </StepperItem>

  <StepperItem title="VScode">
    You can integrate with VS Code to make your workflow faster. To open the project with VS Code, you should download it. <a href="https://code.visualstudio.com/" target="blank">Click to download</a>
  </StepperItem>

  <StepperItem title="Rust">
    (Optional) If you want to code in Tauri and Axum, you need to install Rustup & Cargo on your desktop. <a href="https://www.rust-lang.org/tools/install" target="blank">Click to download</a>
  </StepperItem>

  <StepperItem title="Go">
    (Optional) If you want to build backend services with Gin, you need to install Go on your desktop. <a href="https://go.dev/dl/" target="blank">Click to download</a>
  </StepperItem>

  <StepperItem title="Dose3">
    To use the DosE CLI, run this command in your terminal and enjoy:

    ```shell title="Install using npm"
    npm i -g dose3@latest
    ```
  </StepperItem>
</Stepper>

## Quick Setup with npx

For a faster setup, use the `npx` command to create a new dose3 project in one step:

```plaintext
npx dose3@latest
```
