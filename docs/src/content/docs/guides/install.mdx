<Steps>

1.  #### NodeJS :
    <Icon name="seti:npm" size="2rem"/>
    this cli based on Nodejs npm package and written in Javascript code in this case we need to install nodejs first <a href="https://nodejs.org/en/" target="blank">click to download</a>
2. #### Git :
    <Icon name="seti:git" size="2rem"/>
    in this process we use git repository to clone project to you if you want to use it you should install git <a href="https://git-scm.com/" target="blank">click to download</a>
3. #### VScode
    we can integrate with vscode to make your life faster to open project with vscode you should download vscode to start project <a href="https://code.visualstudio.com/" target="blank">click to download</a>

4. #### Rust 
    <Icon name="seti:rust" size="2rem"/>
    (optional) if you want to code in Tauri and Axum you need to install rustup & cargo in your desktop <a href="https://www.rust-lang.org/tools/install" target="blank">click to download</a>

5. #### install package
    &nbsp;&nbsp;&nbsp;to use dose cli you need to run this command in your therminal then get enjoy

    ```shell title="install using npm"
    npm i -g dose3@latest
    ```
</Steps>
