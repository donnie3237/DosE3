---
title: เริ่มสร้างโปรเจค
description: การแนะนำ การใช้ dose3 เพื่อเริ่มโปรเจค ของคุณ
---

ในคำสั่งของเรา มีการตรวจสอบเวอร์ชั่น อัพเดทเวอร์ชั่น และ เริ่มสร้างโปรเจคดังนี้

- สามารถเช็คเวอร์ชั่นโดยการรันคำสั่ง `dose -v`
- สามารถอัพเดทเวอร์ชั่นโดยการรัน `dose update`
- ในการเริ่มสร้างโปรเจค สามารถรันคำสั่ง `dose` เพื่อเริ่มต้น
