&nbsp;&nbsp;&nbsp; เพื่อเริ่มการติดตั้ง dose3 สามารถรันคำสั่งด้านล่างนี้ แต่ต้องติดตั้ง NodeJS ก่อนนะ

```shell title="install using npm"
npm i -g dose3@latest
```
### NodeJS

&nbsp;&nbsp;&nbsp; dose3 ถูกเขียนขั้นด้วยภาษาJavascript และ เผยแพร่ลงบน node package manager จึงมีความจำเป็นที่จะต้องเริ่มติดตั้ง nodejs ก่อนรันคำสั่ง<a href="https://nodejs.org/en/" target="blank">click to download</a>

### Git
&nbsp;&nbsp;&nbsp;  ในกระบวนการทำงานของเรา ใช้ Git เป็นเบื้องหลังการทำงาน ดังนั้นจึงต้อง ทำการติดตั้ง git ก่อน เพื่อทำให้ระบบสามารถใช้งานได้สมบูรณ์ <a href="https://git-scm.com/" target="blank">click to download</a>

### VScode
&nbsp;&nbsp;&nbsp; —ไม่บังคับ— คุณสามารถรันคำสั่งและเปิด vscode ขึ้นมาในตัวได้โดยการ ติดตั้งมันก่อน ที่จะรันคำสั่งของเรา เพื่อให้การทำงานลื่นไหลมากขึ้น<a href="https://code.visualstudio.com/" target="blank">click to download</a>

### Rust
&nbsp;&nbsp;&nbsp; -ไม่บังคับ- ถ้าเกิดคุณต้องการ เริ่มบางอัน อาจจำเป็นต้องติดตั้ง เช่น Tauri , Actix เป็นต้น<a href="https://www.rust-lang.org/tools/install" target="blank">click to download</a>
